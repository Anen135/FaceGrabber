Справка: Программа для распознавания лиц

Описание

Данная программа выполняет распознавание лиц в режиме реального времени. Она позволяет захватывать лица, сохранять их как изображения и экспортировать в виде файлов PNG.

Основные функции

  • Распознавание лиц: Вы можете запускать и останавливать процесс распознавания лиц.
  • Захват лица: Захватывайте обнаруженные лица для сохранения.
  • Очистка захваченных лиц: Удаляйте все захваченные изображения из программы.
  • Экспорт изображений: Сохраняйте захваченные лица в выбранную папку.

Как использовать

  1. Запустите программу, дважды щелкнув на исполняемый файл.
  2. Нажмите Start Detection, чтобы начать процесс распознавания лиц.
  3. Используйте кнопку Capture Face для сохранения лиц, которые программа обнаружила.
  4. Если хотите удалить все сохранённые лица, нажмите Clear Faces.
  5. Чтобы сохранить лица на диске, используйте меню Экспорт.

Горячие клавиши

  • S: Запуск распознавания лиц.
  • E: Остановка распознавания лиц.
  • C: Захват лица.
  • W: Очистка списка захваченных лиц.
  • Escape: Закрытие приложения.

Рекомендации

Рекомендуемое расстояние до камеры для оптимального распознавания лиц — около 5 метров.

Ввод работает с реальными клавишами, поэтому не забудьте нажать Shift + Alt, чтобы сменить раскалдку клавиатуры на латинские клавиши.
